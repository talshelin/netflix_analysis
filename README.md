# 🎬 Netflix Content Strategy Analysis

## 📌 Business Problem

Analyzing Netflix's content distribution to understand trends and strategic direction.

## 📊 Dataset

Netflix Movies and TV Shows dataset from Kaggle.

## 🧹 Data Preparation

* Handled missing values
* Processed categorical variables
* Extracted genre information

## 📈 Analysis

* Content distribution (Movies vs TV Shows)
* Yearly content production trends
* Country-based production
* Genre analysis

## 🔍 Key Insights

* Movie content dominates the platform
* Content production has increased significantly after 2015
* Drama and Comedy are the most common genres

## 💡 Business Recommendations

* Increase investment in TV series for user retention
* Expand global content production
* Focus on high-performing genres

## 🛠 Tools

Python, Pandas, Matplotlib, Seaborn
